# Whoop + Claude AI Experiment

Personal AI experiment connecting wearable data from Whoop with Claude to explore how LLMs can help interpret recovery, sleep and training signals.

The goal was not to build a production application, but to better understand how APIs, personal data, prompting and AI-assisted analysis can be combined into a useful workflow.

## Why I built it

I wanted to get closer to how AI products are actually built and used, beyond the commercial layer. This project helped me explore how structured personal data can be transformed into more useful insights through an LLM interface.

## What it does

At a high level, the project connects Whoop data with Claude and uses prompts to interpret patterns around recovery, sleep and training readiness.

## What I learned

- How to work with API-based data flows
- How LLMs can turn raw data into more usable explanations
- Where prompting helps, and where structured context is still required
- Why user trust, data privacy and clarity of output matter in AI workflows

## Notes

This is a personal learning project, not a production application.
No personal tokens, credentials or sensitive data are included in this repository.
